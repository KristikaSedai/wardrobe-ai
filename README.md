# wardrobe-ai
Create a virtual environment first: <br>
```
python -m venv venv 
source venv/bin/activate  # Mac/Linux 
venv\Scripts\activate     # Windows 
```
Then install: <br>
```
pip install -r requirements.txt
```
### Dataset 
Fashion product images dataset: https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-dataset/data <br>
Fashion Product Images (Small 280MB) version: https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-small <br>
